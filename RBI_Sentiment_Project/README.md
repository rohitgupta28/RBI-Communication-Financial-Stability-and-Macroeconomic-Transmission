## Project Overview

This project analyzes how the **tone and language** of Reserve Bank of India (RBI) official
communications — specifically Monetary Policy Committee (MPC) minutes — affect key macroeconomic
outcomes in India over the period 2010 to 2025.

The core idea is that the *words* the RBI uses carry information beyond just the policy
decision itself. A hawkish statement (negative, cautious tone) can tighten financial conditions
even before a rate hike happens. A dovish statement (positive, confident tone) can stimulate
credit and equity markets before a rate cut.

The Financial Stability Report (FSR) is used as an **outcome variable** — measuring how
monetary policy tone transmits into the stability of the financial system over time.

### What makes this study unique
- First study to apply the **Loughran-McDonald Finance Dictionary** specifically to RBI MPC corpus
- Covers **15 years** (2010–2025) across multiple monetary policy regimes
- Connects sentiment to **7 macroeconomic variables** using Granger causality and VAR models
- Builds a **Financial Stability Sentiment Index (FSSI)** and **Monetary Policy Stance Index (MPSI)**
- Classifies each MPC period as **Hawkish / Dovish / Neutral**

---

## Research Questions

1. Does the tone of RBI MPC minutes significantly correlate with macroeconomic outcomes
   (CPI inflation, GDP growth, bank credit growth, Nifty 50, USD/INR)?
2. Does MPC communication sentiment **Granger-cause** changes in financial stability (FSSI)?
3. Can the Hawkish/Dovish stance derived from text predict future monetary policy decisions?
4. How did key structural events (COVID-19, demonetisation, MPC framework launch, rate hike cycle)
   affect the tone of RBI communication?

## Methodology

### 1. Sentiment Index Formula

The core sentiment score follows the **Tetlock (2007)** approach, adapted with the
**Loughran-McDonald (2011) Finance Dictionary**:

```
Sentiment Index = (Positive Words − Negative Words) / Total Words
```

- **Positive index value** → Accommodative / Dovish language
- **Negative index value** → Restrictive / Hawkish language
- Dividing by total words normalises for document length

### 2. Financial Stability Sentiment Index (FSSI)

Built from FSR sentiment scores. More negative FSR tone = lower financial stability.
Normalised to a 0–100 scale with 3-period rolling smoothing.

```
FSSI_raw      = −1 × FSR_Sentiment_Index
FSSI          = 100 × (FSSI_raw − min) / (max − min)
FSSI_smoothed = 3-period rolling average of FSSI
```

### 3. Hawkish / Dovish / Neutral Classification

Each MPC period classified by deviation from mean sentiment:

```
Dovish  → Sentiment Index > mean + 0.3 × std
Hawkish → Sentiment Index < mean − 0.3 × std
Neutral → everything in between
```

### 4. Econometric Tests

| Test | Purpose | Library |
|------|---------|---------|
| ADF Unit Root | Check stationarity before regression | statsmodels |
| Pearson Correlation | Linear association between sentiment and macro vars | pandas |
| OLS Regression | Estimate impact of sentiment on each macro variable | statsmodels |
| Granger Causality | Test if past sentiment predicts future macro outcomes | statsmodels |
| VAR Model | Capture bidirectional dynamics between variables | statsmodels |
| Impulse Response | Trace effect of a sentiment shock over time | statsmodels |
| Event Study | Sentiment behaviour around 7 key policy events | custom |

## Notebook Guide

Run notebooks **strictly in order** (01 → 02 → 03 → 04 → 05 → 06).

### Notebook 01 — Setup and Word List
**Purpose:** Install all packages and load the Loughran-McDonald Finance Dictionary.

**Key output:**
- `data/raw/LM_MasterDictionary.csv` — 140 positive + 893 negative finance words

**Why LM dictionary?**
Generic sentiment tools (VADER, TextBlob) are built for social media.
The LM dictionary was built specifically for financial regulatory language —
the exact type of text in RBI documents.

---

### Notebook 02 — Sentiment Extraction
**Purpose:** Download all 62 RBI PDFs and compute sentiment scores using LM dictionary.

**3-layer fallback strategy:**

| Layer | Method | When used |
|-------|--------|-----------|
| Layer 1 | Direct URL download from RBI website | Works on home/college WiFi |
| Layer 2 | Read from local PDFs in `data/raw/pdfs/` | If website blocks automated download |
| Layer 3 | Convert existing scraped CSV data | Fastest fallback if both above fail |

> **Note:** RBI's website blocks automated downloads from cloud/server IPs (returns 403).
> This is normal — Layer 1 works fine on a personal laptop.
> If blocked, use Layer 3 which uses your already-scraped data from the original project.

**Key outputs:**
- `data/processed/mpc_sentiment_lm.csv` — 31 MPC documents scored
- `data/processed/fsr_sentiment_lm.csv` — 31 FSR documents scored
- `data/processed/combined_sentiment_lm.csv` — both combined

### Notebook 05 — Econometric Analysis
**Purpose:** Run all statistical tests and save results tables.

**7 analyses performed:**

**Table 1 — Descriptive Statistics**
Summary statistics (mean, std, min, max) for all 10 variables.

**Table 2 — ADF Unit Root Test**
Tests whether each time series is stationary.
Non-stationary series are first-differenced before regression.

**Table 3 — Correlation Matrix**
Pearson correlations between MPC Sentiment, FSR Sentiment, and all macro variables.

**Table 4 — OLS Regression**
Regresses each macro variable on MPC Sentiment Index.
Reports R², adjusted R², F-statistic, coefficients, and p-values.

**Table 5 — Granger Causality Test**
Tests whether past MPC sentiment values help predict:
CPI Inflation, GDP Growth, Nifty 50, Bank Credit Growth, USD/INR, FSSI

**Table 6 — VAR Model**
Vector Autoregression capturing bidirectional dynamics.
Reports optimal lag order (AIC), model coefficients, and IRF data.

**Table 7 — Event Study**
Sentiment scores around 7 key structural events:
- Taper Tantrum (Jul-2013)
- MPC Framework Launch (Oct-2016)
- Demonetisation Shock (Dec-2016)
- Rate Cut Cycle Start (Jun-2019)
- COVID Emergency Cut (Jun-2020)
- Post-COVID Rate Hike (Jun-2022)
- Rate Cut Restart (Dec-2024)

## Expected Results

Based on existing literature and preliminary data:

- **MPC sentiment and CPI:** Negative correlation expected — more hawkish tone precedes lower inflation
- **MPC sentiment and Nifty 50:** Positive correlation expected — dovish tone boosts equity markets
- **MPC sentiment and FSSI:** Positive correlation expected — confident RBI tone improves stability
- **Granger causality:** MPC sentiment expected to Granger-cause CPI and market variables at lag 1–2
- **Hawkish periods:** 2011–2014 (high inflation cycle), 2022–2023 (post-COVID rate hike)
- **Dovish periods:** 2019–2020 (pre-COVID cuts), 2020–2021 (COVID accommodation)

## Troubleshooting

### RBI PDF download returns 403 error
```
HTTP 403 Forbidden — RBI is blocking this IP/environment
```
**Fix:** Use Layer 3 in Notebook 02 — it will automatically use your existing CSV data.
Alternatively, run the notebook from your home WiFi or college network.

### `figure.constrained_layout` KeyError
```
KeyError: 'figure.constrained_layout is not a valid rc parameter'
```
**Fix:** Replace with `'figure.constrained_layout.use': True` in the rcParams block.
This affects older matplotlib versions (pre-3.6).

### `stats` NameError in Figure 4
```
NameError: name 'stats' is not defined
```
**Fix:** Add `from scipy import stats` to the imports cell (Cell 1) of Notebook 06.

### `OUTPUT_RES` NameError in Figure 7
```
NameError: name 'OUTPUT_RES' is not defined
```
**Fix:** Re-run the imports cell (Cell 1) before running Figure 7.
Or add `OUTPUT_RES = os.path.join(PROJECT_ROOT, 'output', 'results')` at the top of the cell.

### VAR model fails
```
ValueError: sample size is too small to estimate a VAR
```
**Fix:** Reduce `maxlags` parameter. With 31 observations, use `maxlags=2` or `maxlags=3`.

### yfinance download returns empty DataFrame
```
No data found for ^NSEI
```
**Fix:** Check internet connection. Try `yf.download('^NSEI', period='15y', progress=False)`.
